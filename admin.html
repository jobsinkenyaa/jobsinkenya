📖 Manual Upload (if publish button fails)

Download jobs.json and upload it directly to your GitHub repo.

1. Click "Download jobs.json" above
2. Go to github.com/jobsinkenyaa/jobsinkenya
3. Click Add file → Upload files
4. Upload jobs.json → Commit changes

⚙️ Settings

GitHub token & admin login

🔑 GitHub Token (Required for Publish)

Your token is saved only in this browser. Never shared anywhere.

How to generate a token:
1. Go to github.com → Profile picture → Settings
2. Scroll to bottom → Developer settings → Personal access tokens → Tokens (classic)
3. Click Generate new token (classic)
4. Name: anything · Expiration: No expiration
5. Tick only ✅ repo → Scroll down → Generate token
6. Copy it immediately and paste below
